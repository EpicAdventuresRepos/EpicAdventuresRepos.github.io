Synonymous

Pursuing Tom Ram has no synonymous. I had no free memory for them. If you want to add synonymous, and you have enough free Ram, here you have my suggestions.

Create a vector that associates a verb to a number.

Then create a second vector that associates a number to a verb command.

Sadly I was not able to store data or code directly in page 0 using CBM PRG Studio. If you know how I would like to hear you.

Now, you can define verbs like PUT o DROP and associate both with the number 3. Then, you associate number 3 to the DROP_SUBROUTINE.

I have no example code but it should be trivial using this adventure as source.
