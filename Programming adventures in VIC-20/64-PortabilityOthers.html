Portability to other 6502 systems

It should be easy to run Pursuing Tom Ram in other systems with a 6502 processor. In fact, I learnt many interesting things from Atari 800 and Apple II books and source code to make this game.

I did not have time yet to make tests but I would like to try with both systems. If you are interested just let me know.
